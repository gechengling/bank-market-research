# Bank Market Research Assistant
# 银行市场研究助手

> *AI-powered market research and competitive intelligence — analyze industries, map competitive landscapes, size markets, and generate actionable bank strategy reports.*

> AI驱动的市场研究与竞争情报工具——分析行业格局、绘制竞争态势、估算市场规模，生成可操作的银行战略报告。

---

## Overview

This skill provides comprehensive market research capabilities for bank strategy teams and corporate banking officers. It covers industry landscape analysis, Porter's Five Forces, competitive mapping, market sizing, policy assessment, and banking opportunity evaluation.

本工具为银行战略团队和对公业务人员提供全面的市场研究能力，涵盖行业格局分析、波特五力模型、竞争版图绘制、市场规模估算、政策评估及银行机会评估。

## Installation

```bash
clawhub install bank-market-research
```

## Core Capabilities

| Capability | Description |
|------------|-------------|
| Industry Analysis | Landscape, value chain, profit pools, lifecycle |
| Competitive Intelligence | Porter's Five Forces, competitor mapping, benchmarking |
| Market Sizing | TAM/SAM/SOM estimation with multiple methodologies |
| Policy Assessment | Regulatory environment, policy impact on banking |
| Banking Opportunity Matrix | Product fit, customer segmentation, risk-return |
| Regional Risk Analysis | LGFV risk, regional economic assessment |
| Structured Reports | Complete market intelligence reports |
